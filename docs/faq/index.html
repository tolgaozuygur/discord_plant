Is there a limit of giving water to the plant?


No, there is no limit. Because we want to make a project that requires teamwork to grow the plant.

How can I water the plant?


You can water the plant by using the "!water" command.

Can I see the plant?


Yes, you can see the plant by using the "!photo" command. The photo will update every hour.
